Information

Jorinde httpd is a webserver (http 1.0 and some of the important features of 1.1). It also supports WebDAV through a plugin which is in the default installation.

The grand plan is: HTTP/1.[0/1], WebDAV, and usable API's to write extensions.

The server is written in LPC runs (by default) above vanilla DGD and its Kernel Library.

Development started on 20th of January 2004, but due to lazines and being somewhat busy I've taken longer breaks from working on it lately. The current version is 0.3.13 and I intend to release full source-code when I bump the version up to 0.4.0; I expect that to happen this month (july 2004).

The source code will be available from a Subversion repository (a compelling CVS replacement) at Berlios, which will also be the future home of the application.

Grab one of the packaged releases.

Browse the sourcetree in the repository.

Or just check out the latest version using svn:
